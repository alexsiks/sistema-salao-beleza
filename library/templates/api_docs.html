Introdução

A API REST do Sistema de Biblioteca permite acesso programático ao acervo, reservas e usuários. Todos os dados são retornados em formato JSON.

URL Base
{{ base_url }}/api/
Autenticação

Envie o header Authorization em todas as requisições autenticadas:

Authorization: Token <seu_token>
Exemplo com cURL
curl -H "Authorization: Token <seu_token>" \ {{ base_url }}/api/books/
Códigos de Resposta
CódigoSignificado
200OK — Requisição bem-sucedida
201Created — Recurso criado
400Bad Request — Dados inválidos
401Unauthorized — Token ausente ou inválido
403Forbidden — Sem permissão
404Not Found — Recurso não encontrado
