Como funciona o cálculo
1
Solicitação

Usuário solicita empréstimo. O valor do livro e a multa/dia são registrados no momento da solicitação.

2
Confirmação de Retirada

Admin confirma que o livro foi retirado. Data do empréstimo e prazo de devolução são definidos automaticamente (hoje + {{ config.max_loan_days }} dias).

3
Devolução

Admin registra a devolução. Se houver atraso, a multa é calculada automaticamente: dias de atraso × R$ {{ config.fine_per_day }}.

4
Pagamento da Multa

Após cobrar a multa, admin marca como "Multa Paga" no sistema.
